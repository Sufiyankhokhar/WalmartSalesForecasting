# Walmart-Sales-Forecasting
This project focuses on forecasting weekly sales for Walmart stores using Machine Learning techniques. Accurate sales forecasting helps retailers optimize inventory, staffing, and promotional strategies.  The project includes data analysis, feature engineering, model training, and a deployed Streamlit web application for interactive predictions.

# Problem Statement
Retail sales are affected by seasonality, holidays, promotions, and economic factors.
The goal of this project is to predict future weekly sales to support better business decisions.
